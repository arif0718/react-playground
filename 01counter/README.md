# Click-n-Count 🧮

A simple and interactive counter app built with React. This project helps demonstrate the use of React hooks like `useState` for managing state in functional components.

---

## 🚀 Features

- ✅ Increment the count
- ✅ Decrement the count
- ✅ Reset to zero
- ✅ Clean and responsive UI

---

## 🛠️ Tech Stack

- React (useState())
- Tailwind CSS 
- JavaScript (ES6+)
- Shadcn/ui
